JavaScript이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

The html, which is known as the root element, wraps the contents of a web page and it can be found im mediately after the doctype declaration.

Metadata contains information about the page like styles, scripts and data to help search engines or browsers to use and render the page.
